== Description ==

Wp-Feedburner is a plugin which deals with everything Feedburner but for now, it will simply add your feedburner subscriber count to your Dashboard(in Right Now) irrespective of old or new feedburner URL

== Installation ==

1. Activate the plugin through the 'Plugins' menu in WordPress

2. Access Wp-Feedburner settings via the "settings" menu and add your Feedburner Id

3. Your stats should now show up in your Dashboard

== Frequently Asked Questions ==

1. Can I use both (old and new) feedburner Url's ?

   Yes, Both old(starts with http://feeds.feedburner.com) and new (starts with http://feeds2.feedburner.com) are Supported.
